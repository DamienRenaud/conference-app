### Introduction

Application de prise de notes lors de conférences.

L'application a été réalisée dans le cadre du cours de développement multi-plateforme en 3ème année d'école d'ingénieur par alternance à l'IMT Atlantique.


### Fiche technique

Application dévloppée avec Ionic v1.

### Lancement de l'application

#### Sur desktop

Lancer la commande : ionic serve

L'application appelle certaines fonctionnalités des smartphones (contacts, caméra, etc...). 
Il est donc nécessaire d'utiliser des utilitaires comme cordova-mocks sur Chrome pour tout afficher.

#### Sur mobile

1. Brancher le smartphone et passer en mode développeur
2. Lancer la commande : ionic cordova run android (ou ios)
